# React Vite CI/CD Starter 🚀

A **React + Vite starter project** with integrated **CI/CD pipeline**, **ESLint**, and **SonarQube** for code quality.  
This template is designed for modern frontend development, offering fast builds, linting, and automated quality checks.

---

## 📦 Features
- ⚛️ **React + Vite** for fast development and optimized builds
- ✅ **ESLint** for consistent code style and error prevention
- 📊 **SonarQube integration** for maintainability and security analysis
- 🔄 **CI/CD pipeline ready** (GitHub Actions / Jenkins / other tools)
- 📂 Structured project layout (`src/`, `public/`, configs) for easy scalability

---

## 🛠️ Getting Started

### Prerequisites
- Node.js (>= 18.x recommended)
- npm or yarn
